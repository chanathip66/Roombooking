# 🏢 MeetRoom — ระบบจองห้องประชุมของหน่วยงานภายในองค์กร

เว็บแอปพลิเคชันสำหรับจองห้องประชุมภายในองค์กร แยกสิทธิ์ผู้ใช้ 2 กลุ่ม ได้แก่ **ผู้ใช้งานที่ต้องการจองห้องประชุม** และ **ผู้ดูแลระบบ** พร้อมตารางเวลาแบบไทม์ไลน์ ดีไซน์สไตล์ Paper-like เรียบง่าย อ่านง่าย เหมือนสมุดจองห้องประชุมในสำนักงานจริง

---

## ✨ คุณสมบัติหลัก

- 🔐 **ระบบลงทะเบียน / เข้าสู่ระบบ** — ผู้ที่ต้องการจองต้องลงทะเบียนและเข้าสู่ระบบด้วยชื่อผู้ใช้ + รหัสผ่านก่อนใช้งาน
- 🛠️ **โหมดผู้ดูแลระบบ** — เข้าใช้งานด้วยรหัสผู้ดูแล สามารถดูและจัดการ (ยกเลิก) การจองได้ทุกรายการ
- 📅 **ตารางเวลาแบบไทม์ไลน์** — แกนนอนเป็นเวลา 08:00–18:00 น. แกนตั้งเป็นห้อง เห็นภาพรวมการใช้ห้องทั้งวันในหน้าเดียว
- 📝 **ฟอร์มจองแบบ Slide-in Panel** — คลิกช่องว่างในตารางเพื่อเริ่มจอง ชื่อ/หน่วยงานถูกเติมอัตโนมัติจากบัญชีผู้ใช้
- ⛔ **ป้องกันการจองซ้ำทับเวลา** — ตรวจสอบช่วงเวลาทับซ้อนทั้งฝั่งหน้าเว็บและฝั่งเซิร์ฟเวอร์
- ❌ **ยกเลิกการจองตามสิทธิ์** — ผู้จองยกเลิกได้เฉพาะของตนเอง / ผู้ดูแลยกเลิกได้ทุกรายการ
- 🌗 **Light / Dark Mode** — สลับธีมได้จริง จำค่าไว้ในเครื่อง
- 💀 **Skeleton Loader** — แสดงโครงตารางระหว่างกำลังดึงข้อมูล
- 📜 **บันทึกประวัติ (Logs)** — ทุกการสร้าง/ยกเลิกการจองถูกบันทึกลงฐานข้อมูล

---

## 🧱 เทคโนโลยีที่ใช้

| ส่วน | เทคโนโลยี |
|---|---|
| Frontend | HTML + Tailwind CSS (CDN) + Vanilla JavaScript |
| Backend | Google Apps Script (RESTful API — `doGet` / `doPost`) |
| Database | Google Sheets (4 ชีต: `Rooms`, `Bookings`, `Admins`, `Logs`) |
| ฟอนต์ | IBM Plex Serif / IBM Plex Sans Thai / IBM Plex Mono |

---

## 🗂️ โครงสร้างฐานข้อมูล (Google Sheets)

- **Rooms** — `RoomID, RoomName, Capacity, Location, Equipment, Status, Note`
- **Bookings** — `BookingID, RoomID, BookingDate, StartTime, EndTime, Topic, Attendees, BookerName, BookerEmail, Status, ReminderSent, CreatedAt, UpdatedAt, CancelledBy`
- **Admins** — `Email, Name, Status, AddedAt`
- **Logs** — `LogID, Timestamp, BookingID, Action, PerformedBy, Detail`

## 🚀 วิธีใช้งาน

1. เปิดไฟล์ `index.html` ด้วยเว็บเบราว์เซอร์ (แนะนำ Chrome / Edge)
2. **ผู้จอง:** ลงทะเบียนบัญชีใหม่ หรือเข้าสู่ระบบด้วยบัญชีทดลอง `usersau / 123456`
3. คลิกช่องว่างในตาราง หรือปุ่ม **"จองห้องประชุม"** → กรอกรายละเอียด → กด **"ยืนยันการจอง"**
4. **ผู้ดูแล:** จากหน้าเข้าสู่ระบบ กดลิงก์ **"สำหรับผู้ดูแลระบบ"** → ป้อนรหัส `dtiadminsau`
